# press logos

Drop SVG or PNG logos here for the "Featured in" strip on the homepage.

## Currently text wordmarks

The strip renders as styled text (CIO Magazine / SAP Press / Wiley
For Dummies) until real logos land here. When they do, I'll swap the
text spans for `<img>` tags.

## Filename convention

Use the publication's slug, lowercase, with extension:

- `cio.svg`
- `sap-press.svg`
- `wiley.svg`

SVG preferred. PNG fine if the original is raster (export at ~2x for
retina, optimize, target under 50KB each).

## Heads-up

Public repo. Don't upload anything you don't have rights to display.
Standard "as featured in" attribution is fair use for these
publications.
