# assets

Drop the headshot here.

## How to upload (Ian)

1. Go to: https://github.com/IJKMan/iankimbell.com/tree/main/assets
2. Click **Add file → Upload files**
3. Drag the photo in
4. Filename should be exactly `ian.jpg`
   (or `ian.png` / `ian.webp` — let Mike know if it's not `.jpg` so the CSS reference updates)
5. Commit directly to `main`

The site auto-deploys via Cloudflare Pages within ~60 seconds of the commit.

## Photo guidance

- **Orientation:** portrait (taller than wide). The hero column is tall and narrow.
- **Resolution:** at least 1200 × 1800 px. Larger is fine; Cloudflare will resize.
- **Format:** JPG or WebP preferred. Aim for under ~500 KB after export.
- **Crop:** leave a little headroom; the photo runs full-bleed on the right side of the hero.
- **Style:** professional but warm — the kind of shot a senior buyer would expect on
  the page of someone they might book for a keynote.
