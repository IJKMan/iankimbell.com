# draft

A scratch folder for Ian to drop raw materials that should land on the site
but aren't ready as final copy yet.

## What goes here

- Screenshots of slides, past keynotes, awards, anything visual you might
  want incorporated
- Text drafts: bio paragraphs, service descriptions in your own voice,
  testimonial quotes you've collected, taglines you're trying out
- PDFs, Word docs, plain text — anything is fine
- Reference material: links, articles, things you want Mike + Claude to look at

Nothing in this folder appears on the website itself. It's only here so Mike
and Claude can read it and integrate the substance into the live page.

## How to upload (Ian)

1. Go to: https://github.com/IJKMan/iankimbell.com/tree/main/draft
2. Click **Add file → Upload files**
3. Drag whatever you have in
4. Scroll down and click the green **Commit changes** button to finalize

## Heads-up on privacy

This is a **public** GitHub repository — anyone with the URL can browse it
and read anything in `draft/`. Don't upload anything you wouldn't want
on the open internet: confidential client names under NDA, contracts,
personal addresses, financial details, etc. If something is sensitive,
share it with Mike directly instead.

## Naming

No naming convention required — drop files in however you like. If you want
to leave a note about what something is, put it in the commit message or
add a short text file alongside the upload.
